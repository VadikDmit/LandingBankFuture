# Техническое Задание (ТЗ) на разработку сайта BankFuture

## 1. Общие сведения
**Название проекта:** BankFuture
**Цель проекта:** Создание корпоративного сайта для компании BankFuture, занимающейся разработкой передовых решений для финансового рынка.
**Стек технологий:**
- Frontend: React + Vite
- Хостинг/Deployment: Vercel

## 2. Структура и контент
Сайт должен презентовать основные продукты компании.

### 2.1. Главная страница
- Презентация компании BankFuture как лидера в финтех-разработке.
- Навигация по продуктам.

### 2.2. Продукты (Основные разделы)

#### А. Искусственный интеллект - консультант для НПФ
- **Описание:** Решение для НПФ (в частности, НПФ Ростех).
- **Функционал:** Интеллектуальный консультант на сайте фонда.

#### Б. Личный кабинет финансового консультанта
- **Описание:** Рабочее место для профессионалов.
- **Особенности:** Интеграция инструментов финансового планирования.

#### В. Конструктор чат-ботов для агентов
- **Описание:** ИИ-ассистент для агентов и финансовых консультантов.
- **Функционал:** Возможность гибкой настройки ботов под нужды агента.

#### Г. ИИ Юрист
- **Описание:** Лучший юридический ИИ на российском рынке.
- **Уникальное преимущество:** Возможность локального запуска (конфиденциальность и автономность).

## 3. Требования к дизайну (в разработке)
- Современный, технологичный стиль.
- Адаптивность (Desktop/Mobile).
- Акцент на инновационность и надежность.
